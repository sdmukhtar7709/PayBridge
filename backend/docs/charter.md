# Project Charter

**Problem (short):** Easy, trusted way to convert between digital (UPI) and cash via nearby verified agents.

**Objectives (MVP):**
- Let users request cash-in or cash-out.
- Discover nearby verified agents.
- Secure verification (OTP/QR) and transaction logging.
- Simple admin approval for agents.

**Success Criteria:**
- End-to-end happy path works on a test device.
- Nearby search responds in <2s in a test area.
- OTP verification flow works and records transactions.
