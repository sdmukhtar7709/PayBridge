# User Stories (MVP)

- As a **User**, I can register/login.
- As a **User**, I can request cash-out (get cash) or cash-in (deposit cash) by entering amount and sharing location.
- As a **User**, I can view a matched/nearby agent and the status of my request.
- As a **User**, I can verify the transaction using OTP/QR and mark it complete.
- As an **Agent**, I can set availability, limits, and accept/reject a request.
- As an **Agent**, I can see OTP/QR and mark a transaction completed.
- As an **Admin**, I can approve/ban agents (manual toggle).
